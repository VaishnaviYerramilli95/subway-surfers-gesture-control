# 🎮 Subway Surfers Gesture Control
**NNDL Project** — Real-time hand gesture recognition using MediaPipe + TensorFlow/Keras + Flask

[![Live Demo](https://img.shields.io/badge/Live_Demo-Render-blueviolet?style=for-the-badge)](https://subway-surfers-gesture-control.onrender.com)

---

## 🧠 How It Works

```
Webcam → MediaPipe Hands → 63 Landmarks → Neural Network → Gesture → Keypress
```

1. **MediaPipe Hands** extracts 21 hand landmarks (63 x/y/z coordinates) per frame in the browser
2. Landmarks are sent to a **Flask API** running the trained **Keras neural network**
3. The model classifies the gesture into one of 5 classes
4. Locally, `game_controller.py` converts gestures → **keyboard inputs** for Subway Surfers

---

## 🖐 Gestures

| Gesture | Action | Key |
|---------|--------|-----|
| ✋ Open palm raised | Jump | ↑ |
| 👊 Closed fist lowered | Slide / Duck | ↓ |
| 👈 Hand pointing left | Move Left | ← |
| 👉 Hand pointing right | Move Right | → |
| 🖐 Neutral / relaxed | No action | — |

---

## 🏗 Neural Network Architecture

```
Input (63)
    ↓
Dense(128) + BatchNorm + ReLU + Dropout(0.4)
    ↓
Dense(64)  + BatchNorm + ReLU + Dropout(0.3)
    ↓
Dense(32)  + ReLU
    ↓
Dense(5, softmax)   ← 5 gesture classes
```

- **Optimizer**: Adam (lr=0.001)
- **Loss**: Sparse Categorical Crossentropy
- **Training data**: 4,000 synthetic samples (800 × 5 gestures) + Gaussian noise augmentation

## 🚀 Quick Start

### 1. Install dependencies

```bash
pip install -r requirements.txt

# For local game control (not needed on Render):
pip install mediapipe opencv-python pyautogui
```

### 2. Train the model

```bash
python train_model.py
```

This generates:
- `gesture_model.h5` — Saved Keras model
- `label_encoder.pkl` — Gesture class list
- `training_history.json` — Training accuracy/loss curves

### 3. Run the web app locally

```bash
python app.py
# → http://localhost:5000
```

### 4. Control the game (local only)

```bash
# Open Subway Surfers in browser, then:
python game_controller.py
```

## ☁️ Deploy on Render

1. **Push to GitHub:**
   ```bash
   git init
   git add .
   git commit -m "Initial commit — NNDL gesture control project"
   git remote add origin https://github.com/YOUR_USERNAME/REPO_NAME.git
   git push -u origin main
   ```

2. **Connect to Render:**
   - Go to [render.com](https://render.com) → New → Web Service
   - Connect your GitHub repo
   - Render auto-detects `render.yaml` — just click **Deploy**

3. Your live URL will be: `https://subway-surfers-gesture-control.onrender.com`

> **Note:** Free tier sleeps after 15 mins of inactivity (wakes in ~30s).
